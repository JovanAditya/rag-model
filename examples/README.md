# Examples - Academic RAG

Usage examples for the Academic RAG Model.

## Files

| File | Description |
|------|-------------|
| `basic_usage.py` | Basic model usage example |
| `api_integration.py` | FastAPI integration example |
| `evaluation_usage.py` | Evaluation guide for research |
| `golden_set_manager.py` | Golden set management for evaluation |
| `domain_specific_questions.py` | Academic question bank for testing |

## Quick Start

```bash
cd examples

# Basic usage
python basic_usage.py

# API integration example
python api_integration.py

# Evaluation guide
python evaluation_usage.py
```
